### Dependencies

-	[alpine (*3.97MB*\)](https://index.docker.io/_/alpine/)

### Requirements

-	Docker

### Install

```bash
$ git clone https://github.com/maliceio/malice-api
$ cd malice-api
$ docker-compose up -d
```

Navigate to `http://localhost:8080`
